# Digital CV

Welcome to the Digital CV project (Assignment 02)! This project is a modern, responsive digital curriculum vitae built using web technologies.

## 🚀 Live Demo

The project has been automatically deployed and can be viewed here:
**[Live Deployment URL](https://duongnvfx85237.github.io/digital-cv/)**

## 🛠️ Technologies Used

- **HTML5**: For project structure and semantics.
- **CSS3**: For custom styling and component layout (`mobile.css`, `tablet.css`, `laptop.css`).
- **Bootstrap 5**: Utilized the Bootstrap framework (v5.0.2) native grid systems and utilities for fluid components.
- **JavaScript**: For interactive features (Form validation, View more buttons).

## 📂 Project Structure

- `index.html`: The main CV page containing detailed sections including Biography, Job History, Certifications, and Projects.
- `digital-cv.html`: Contains additional embedded project-specific layouts and dashboard outlines.
- `script.js`: Handles interaction scripts.
- `css/`: Dedicated folder for stylesheets defining responsive rules.
- `images/`: Source media items for avatars, thumbnails, and banners.


## 👨‍💻 Installation & Usage

To view the project locally instead of via the live demo:
1. Clone this repository or download the source files.
2. Open `index.html` within any modern web browser to access the website.
